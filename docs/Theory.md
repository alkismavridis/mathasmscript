# VARIABLES

```
v.VAR
[] <--3--> [ a ]

v.A
] <--3--> a ]

v.B
] <--3--> b ]

...
```


# BOOLEAN
```
b.TRUE
b [ x ] --2--> true

b.FALSE
b [ x ] --2--> true

b.NOT
b [ x ] --2--> ! [ x ]

b.OR
b [ x ] --2--> ( b [ x ] && b [ y ] )
```


# NUMBER
```
n.ONE
n [ x ] --2--> true

n.TWO
n [ x ] --2--> false
```

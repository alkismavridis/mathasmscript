# 1. All symbols are created equal.

# 2. Replace stuff with other stuff

- Expression: a sequence of symbols
- Statement: A pair of expression

Examples:
```
2 <----> ( 1 + 1 )
( x + y ) <----> ( y + x )
...
```

# 3. Rename variables
```
x <--3--> y
x <--3--> z
...
```
# 4. Replace variables with expression
```
x --2--> ( x + y )
...
```
